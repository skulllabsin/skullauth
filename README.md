## 🚀 `> QUICKSTART --cpp`

### Step 1 — Include Skull Auth

```cpp
#include "skullauth.h"

// Initialize Skull Auth
SkullAuth auth("YOUR_APP_SECRET", "YOUR_APP_VERSION");
```

### Step 2 — Authenticate User

```cpp
// Authenticate with license key
bool result = auth.license("USER-LICENSE-KEY-HERE");

if (result) {
    // ✅ Auth success — launch your app
    std::cout << "[💀] Welcome, " << auth.user_data.username << std::endl;
    std::cout << "[⚡] Expires: " << auth.user_data.expiry << std::endl;
} else {
    // ❌ Auth failed
    std::cout << "[✗] " << auth.error_message << std::endl;
    exit(1);
}
```

### Step 3 — HWID Lock (Optional)

```cpp
// Lock to hardware ID automatically
auth.set_hwid_lock(true);

// Get current HWID
std::string hwid = auth.get_hwid();
std::cout << "[🖥️] HWID: " << hwid << std::endl;
```

## 🌐 `> QUERY --api --endpoints`

### Base URL
```
https://api.skulllabs.in/v1/
```

### Endpoints

```http
POST /auth/license
Content-Type: application/json

{
  "app_secret" : "YOUR_SECRET",
  "license"    : "USER-KEY-XXXX-XXXX",
  "hwid"       : "HARDWARE-ID-HERE",
  "version"    : "1.0.0"
}
```

**Response — Success:**
```json
{
  "status"   : "success",
  "username" : "SkullUser",
  "expiry"   : "2025-12-31",
  "hwid"     : "locked",
  "level"    : 1
}
```

**Response — Failed:**
```json
{
  "status"  : "error",
  "message" : "Invalid license key"
}
```
